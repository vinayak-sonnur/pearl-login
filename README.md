# Pearl Login System

A simple authentication system built using Next.js API routes with encrypted password handling.

## Features
- User Signup
- User Login
- Password Encryption (bcrypt)
- Token-based authentication
- Protected Dashboard

## Tech Stack
- Next.js
- React
- Node.js
- JavaScript
